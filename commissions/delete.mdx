---
title: "Delete Commission"
api: "DELETE https://dev-api-v.tolt.io/v1/commissions/:id"
description: "This endpoint deletes a commission."
---

## Base URL `https://dev-api-v.tolt.io`

<br />

### Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties" defaultOpen={true}>
    <ResponseField name="id" type="string">
      Unique identifier for the deleted commission
    </ResponseField>
    <ResponseField name="deleted" type="boolean">
      Indicates that the commission was deleted
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X DELETE 'https://dev-api-v.tolt.io/v1/commissions/comm_eK9bzRGn46BhVgNFHD6fDgXW' \
-H 'Authorization: Bearer <API_KEY>'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true,
  "data": {
    "id": "comm_eK9bzRGn46BhVgNFHD6fDgXW",
    "deleted": true
  }
}
```

</ResponseExample>
