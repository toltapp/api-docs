## Base URL `https://api.tolt.io`

<br />
### Body

<ParamField body="referral*" type="string">
  The referral email address or referral ID.
</ParamField>

<RequestExample>

```bash cURL
curl -X POST 'https://api.tolt.io/private/v1/referrals/cancel' \
-H 'Authorization: <API_ID>' \
-H 'Content-Type: application/json' \
-d '{
    "referral": "test@gmail.com",
}'
```
