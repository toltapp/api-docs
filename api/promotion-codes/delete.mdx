---
title: "Delete Promotion Code"
api: "DELETE https://api.tolt.com/v1/promotion-codes/:id"
description: "This endpoint deletes a promotion code."
---

## Base URL `https://api.tolt.com`

<br />

### Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties" defaultOpen={true}>
    <ResponseField name="id" type="string">
      Unique identifier for the deleted promotion code
    </ResponseField>
    <ResponseField name="deleted" type="boolean">
      Indicates that the promotion code was deleted
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X DELETE 'https://api.tolt.com/v1/promotion-codes/prc_dK9bzRGn46BhVgNFHD6fDgXW' \
-H 'Authorization: Bearer <API_KEY>'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true,
  "data": {
    "id": "prc_dK9bzRGn46BhVgNFHD6fDgXW",
    "deleted": true
  }
}
```

</ResponseExample>
